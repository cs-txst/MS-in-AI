# Master of Science in Artificial Intelligence
### Department of Computer Science, Texas State University

---

Welcome to the official repository for the Master of Science in Artificial Intelligence (MS in AI) at Texas State University. This program offers a cutting-edge curriculum designed to prepare graduates for high-impact roles in the rapidly evolving world of AI. The program provides a deep understanding of generative AI, machine learning, data analytics, and ethical AI applications, blending foundational theory with advanced, hands-on skills.

This repository serves as a central resource for prospective students, current students, and faculty.
